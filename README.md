# wisdom-library
<br />

## ✅ 프로젝트 소개
  스프링 MVC 기반의 도서관리 시스템으로, 관리자와 회원으로 구분하여 이용할 수 있도록 페이지를 제작했습니다.
<br /><br />

## ⏰ 프로젝트 기간
  2022년 02월 28일 ~ 03월 07일 (7일)
<br /><br />

## 🧑‍🤝‍🧑 멤버 구성
  개인 프로젝트입니다.
<br /><br />

## ⚙ 개발 환경
  - `eclipse 4.20.0 `
  - `Java`
  - `jdk 11.0.14`
  - `springboot 2.6.4`
  - `mysql 8.0`
